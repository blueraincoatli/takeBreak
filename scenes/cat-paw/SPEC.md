# SPEC: cat-paw（猫咪爪子）

## 概述

一只完整的猫咪前臂（类似泰坦尼克号 Rose 拍在玻璃窗的手掌梗）从屏幕右侧拍过来，卡通风格：粗粗的外轮廓线 + 粉粉的脚掌，视觉冲击力强。

---

## 目标定义

### 核心目标
创建一个趣味互动型休息提醒场景，通过猫咪爪子的"拍打"动作，以轻松可爱的方式提醒用户起身活动。

### 成功标准（Definition of Done）

| 标准 | 检测方法 |
|------|---------|
| 猫爪从屏幕边缘滑入并"拍打"屏幕 | 视觉确认动画流畅播放 |
| 拍打时屏幕出现爪印痕迹 | 视觉确认爪印效果正确渲染 |
| 拍打时播放合成音效（喵声或拍打声） | 听觉确认音效触发 |
| 显示随机提醒文字 | 确认文字内容随机且可见 |
| 点击任意位置可关闭场景 | 点击测试关闭功能 |
| duration 设置为 6-8 秒，自动关闭正常 | 计时确认自动关闭 |
| 在不同分辨率屏幕上显示正常 | 1920x1080 / 2560x1440 测试 |

---

## 技术方案

### 实现方式
- **SVG 动画** — 猫爪使用 SVG 绘制，CSS animation 控制动作
- **爪印效果** — CSS clip-path 或 SVG path 实现爪印形状，配合淡入动画
- **合成音效** — Web Audio API 合成"拍打声"或简短"喵"声

### 动画时序设计

```
0.0s - 屏幕半透明遮罩淡入
0.3s - 猫爪从右侧滑入
0.8s - 猫爪"拍下"动作
1.0s - 爪印出现 + 音效播放
1.5s - 猫爪收回
2.0s - 提醒文字淡入
5.0s - 场景淡出准备关闭
6.0s - 自动关闭
```

### 关键动画

1. **猫爪入场** — 从右侧平移 + 旋转，模拟伸出动作
2. **拍打** — 猫爪快速向下"按"，产生弹性回弹
3. **爪印** — 在猫爪落下位置显示爪印轮廓，渐隐消失
4. **文字** — 随机选择一条提醒文案，居中显示

## 失败处理

| 问题类型 | 处理方案 |
|---------|---------|
| 动画卡顿 | 简化 SVG 路径，减少动画元素 |
| 音效无法播放 | 检查 AudioContext 兼容性，添加用户交互触发 |
| 爪印位置错误 | 固定爪印相对于猫爪的位置 |
| 文字不可读 | 调整颜色/阴影，增强对比度 |
| 自动关闭失效 | 检查 manifest.json 的 duration 字段 |
