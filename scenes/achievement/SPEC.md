# SPEC: achievement（成就解锁）

## 概述

模拟游戏成就解锁界面，以"恭喜解锁久坐大师成就"的反讽方式，幽默地提醒用户休息。

---

## 目标定义

### 核心目标
创建一个游戏化型休息提醒场景，通过游戏成就的视觉语言，以反讽幽默的方式提醒用户休息。

### 成功标准（Definition of Done）

| 标准 | 检测方法 |
|------|---------|
| 成就弹窗视觉符合游戏风格 | 视觉确认与游戏成就相似 |
| 显示"成就图标"和"成就名称" | 确认元素完整 |
| 成就名称幽默反讽（如"久坐大师"） | 确认文案内容 |
| 播放成就解锁音效 | 听觉确认 |
| 显示"奖励"文字（健康的身体） | 确认文字可见 |
| 有闪光的装饰效果 | 视觉确认特效 |
| 点击任意位置可关闭场景 | 点击测试 |
| duration 设置为 8 秒 | 计时确认 |

---

## 技术方案

### 实现方式
- **CSS 动画** — 弹窗弹出、闪光效果
- **SVG 图标** — 成就徽章/奖杯
- **粒子效果** — 金色闪光/星星

### 动画时序设计

```
0.0s - 屏幕半透明暗色遮罩淡入
0.3s - 成就弹窗从屏幕中心弹出（带弹性）
0.5s - 闪光特效播放
0.5s - 成就音效播放
1.0s - 成就图标发光动画
1.5s - 文字依次显示
5.0s - 显示"真正奖励"提示
7.0s - 弹窗淡出准备关闭
8.0s - 自动关闭
```

### 视觉设计

**弹窗样式**：
- 圆角矩形，渐变边框
- 金色/橙色为主色调
- 类似 Steam/PlayStation/Xbox 成就风格

**成就图标**：
- 奖杯或徽章形状
- 金色，带闪光效果
- 可加入"警告"元素增加反讽感

**闪光效果**：
- 从成就图标向外发散的光线
- 或围绕图标旋转的星星

## 检测方法

### 自动化检测
```bash
curl http://localhost:3721/scenes | grep achievement
```

### 手动检测清单
- [ ] 弹窗视觉效果有"游戏感"
- [ ] 音效有"成就解锁"的感觉
- [ ] 文案幽默恰当
- [ ] 整体体验"有趣"而非"讽刺"
- [ ] 点击关闭响应及时

---

## 失败处理

| 问题类型 | 处理方案 |
|---------|---------|
| 不像游戏成就 | 参考真实游戏截图，增加细节 |
| 音效不对 | 调整音调，使用上升音阶 |
| 不够幽默 | 调整文案，增加反差 |
| 过于讽刺 | 软化语气，增加关怀感 |
