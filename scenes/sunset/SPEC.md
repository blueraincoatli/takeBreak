# SPEC: sunset（日落提醒）

## 概述

屏幕渐变为温暖的日落色调，配合宁静的夕阳动画，以温柔治愈的方式提醒用户休息。

---

## 目标定义

### 核心目标
创建一个温馨治愈型休息提醒场景，通过日落视觉效果和柔和的动画，营造放松氛围，引导用户休息。

### 成功标准（Definition of Done）

| 标准 | 检测方法 |
|------|---------|
| 屏幕从当前状态渐变为日落色调 | 视觉确认颜色过渡流畅 |
| 太阳动画从屏幕上方缓慢下落到地平线 | 视觉确认太阳移动轨迹 |
| 天空颜色随太阳位置变化 | 视觉确认天空渐变效果 |
| 显示温馨提醒文字 | 确认文字可读且有氛围感 |
| 点击任意位置可关闭场景 | 点击测试关闭功能 |
| duration 设置为 10 秒，动画完整播放 | 计时确认 |
| 音效（可选）为柔和的环境音或提示音 | 听觉确认（如有） |

---

## 技术方案

### 实现方式
- **CSS 渐变动画** — 天空颜色使用 linear-gradient + CSS 动画过渡
- **SVG 太阳** — 简单圆形 + 光晕效果
- **云朵** — CSS 或 SVG 实现飘动效果（可选）

### 动画时序设计

```
0.0s - 开始：天空渐变淡入（从屏幕截图过渡到日落色）
1.0s - 太阳开始缓慢下落
3.0s - 天空颜色从金黄过渡到橙红
6.0s - 太阳接近地平线，天空渐变为紫红色
7.0s - 提醒文字淡入
9.0s - 场景开始淡出
10.0s - 自动关闭
```

### 颜色方案

```
阶段1（开始）:
  天空: #87CEEB (天蓝) → #FFD700 (金黄)
  太阳: #FFD700 (金黄)

阶段2（中段）:
  天空: #FF8C00 (深橙) → #FF6347 (番茄红)
  太阳: #FF8C00 (深橙)

阶段3（结束）:
  天空: #8B008B (深紫) → #4B0082 (靛蓝)
  太阳: 沉入地平线，只留余晖
```

## 检测方法

### 自动化检测
```bash
# 检查场景被识别
curl http://localhost:3721/scenes | grep sunset
```

### 手动检测清单
- [ ] 天空颜色过渡自然
- [ ] 太阳动画平滑
- [ ] 整体氛围"温馨"而非"压抑"
- [ ] 文字可读且有美感
- [ ] 点击关闭响应及时

---

## 失败处理

| 问题类型 | 处理方案 |
|---------|---------|
| 颜色过渡生硬 | 增加过渡阶段，使用更细腻的 gradient |
| 太阳动画卡顿 | 使用 transform 替代 top/left 移动 |
| 氛围不够温馨 | 调整颜色方案，添加暖色调 |
| 文字不清晰 | 增加文字阴影/光晕 |
