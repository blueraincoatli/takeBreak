# SPEC: fake-bsod（假蓝屏警告）

## 概述

模拟 Windows 蓝屏死机界面，但内容是幽默的"久坐异常"警告，在用户紧张后揭示玩笑，以戏剧性方式提醒休息。

---

## 目标定义

### 核心目标
创建一个搞笑夸张型休息提醒场景，通过"吓一跳"的体验，让用户在紧张后释放，留下深刻印象。

### 成功标准（Definition of Done）

| 标准 | 检测方法 |
|------|---------|
| 初始界面高度还原 Windows 蓝屏风格 | 视觉确认与真实 BSOD 相似度 |
| 错误代码使用幽默命名（如 SIT_TOO_LONG） | 确认错误代码内容 |
| 3 秒后"揭示"动画，显示真实提醒 | 计时确认揭示时机 |
| 揭示时播放轻松音效或"ding"声 | 听觉确认 |
| 显示真正的休息提醒文字 | 确认文字内容清晰 |
| 点击任意位置可关闭场景 | 点击测试 |
| duration 设置为 8 秒 | 计时确认 |

---

## 技术方案

### 实现方式
- **像素风格** — 使用等宽字体模拟终端/BSOD 风格
- **两阶段动画** — 先显示假 BSOD，再揭示真相
- **CSS 过渡** — 实现从严肃到轻松的转变

### 动画时序设计

```
0.0s - 假 BSOD 界面瞬间出现（无淡入，模拟真实崩溃）
0.0-3.0s - 假 BSOD 显示，QR 码区域闪烁
3.0s - "正在收集错误信息..." 进度条动画
4.0s - 界面开始"抖动"
4.5s - 蓝屏"碎裂"或"融化"动画
5.0s - 显示真实提醒界面
5.5s - 播放轻松音效
7.0s - 场景开始淡出
8.0s - 自动关闭
```

### 界面元素

**假 BSOD 阶段**：
```
:(

你的电脑遇到问题，需要重新启动。

我们正在收集一些错误信息，然后为你重新启动。

0% 完成

停止代码：SIT_TOO_LONG_EXCEPTION

详细信息：
- 久坐时间：超过 1 小时
- 腰椎压力：过高
- 血液循环：需要改善

请站起来活动以修复此问题
```

**揭示阶段**：
```
开玩笑的 😄

但说真的，你已经坐了很久了！
站起来活动一下吧～
```

## 检测方法

### 自动化检测
```bash
curl http://localhost:3721/scenes | grep fake-bsod
```

### 手动检测清单
- [ ] 假 BSOD 足够逼真（第一眼）
- [ ] 揭示时机合适（不会太早暴露）
- [ ] 转换动画流畅有趣
- [ ] 整体体验"惊喜"而非"恼人"
- [ ] 文案幽默恰当

---

## 失败处理

| 问题类型 | 处理方案 |
|---------|---------|
| 不够逼真 | 参考 Windows 10/11 真实 BSOD 截图 |
| 揭示太快 | 延长假 BSOD 显示时间 |
| 动画不够戏剧性 | 增加更多特效（抖动、闪屏） |
| 过于吓人 | 缩短假 BSOD 时间，早点揭示 |

```
久坐检测模块发现异常：
- 连续坐姿时间：> 60 分钟

- 预计修复时间：5 分钟活动

按任意键继续...开玩笑的，请站起来
```
