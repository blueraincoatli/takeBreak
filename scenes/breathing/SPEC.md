# SPEC: breathing（呼吸引导）

## 概述

屏幕中央显示一个缓慢膨胀和收缩的圆圈，配合呼吸节奏的文字引导，帮助用户进行短暂的呼吸放松练习。

---

## 目标定义

### 核心目标
创建一个极简优雅型休息提醒场景，通过视觉引导帮助用户进行 1-2 个完整的呼吸周期，达到放松和休息的目的。

### 成功标准（Definition of Done）

| 标准 | 检测方法 |
|------|---------|
| 圆圈平滑膨胀和收缩动画 | 视觉确认动画流畅 |
| 动画周期与呼吸节奏匹配（4-6秒/周期） | 计时确认 |
| 呼吸阶段文字提示（吸气/屏息/呼气） | 确认文字同步显示 |
| 背景颜色随呼吸微妙变化 | 视觉确认颜色变化 |
| 完整执行 2 个呼吸周期后显示提醒 | 计数确认 |
| 点击任意位置可关闭场景 | 点击测试 |
| duration 设置为 15 秒（约 2-3 个周期） | 计时确认 |
| 合成柔和的提示音（可选） | 听觉确认（如有） |

---

## 技术方案

### 实现方式
- **CSS 动画** — 圆圈缩放使用 CSS animation
- **JS 状态机** — 控制呼吸阶段切换和文字更新
- **颜色渐变** — 背景颜色随呼吸阶段变化

### 呼吸节奏设计

采用 4-7-8 呼吸法变体：
```
吸气（4秒）→ 屏息（4秒）→ 呼气（6秒）= 14秒/周期

简化版（更适合初学者）：
吸气（4秒）→ 屏息（2秒）→ 呼气（4秒）= 10秒/周期
```

### 动画时序设计（单周期 10 秒）

```
0.0s - 圆圈最小状态
0.0-4.0s - 吸气：圆圈膨胀到最大
4.0-6.0s - 屏息：圆圈保持最大，微微脉动
6.0-10.0s - 呼气：圆圈收缩到最小
10.0s - 第二个周期开始...
```

### 视觉设计

**圆圈样式**：
- 颜色：柔和的蓝色/紫色渐变
- 边缘：柔和光晕
- 大小：屏幕宽度的 30-50%

**背景变化**：
- 吸气：背景变亮
- 屏息：背景稳定
- 呼气：背景变暗

## 失败处理

| 问题类型 | 处理方案 |
|---------|---------|
| 动画节奏不对 | 调整 animation-duration |
| 圆圈不够柔和 | 增加模糊/光晕效果 |
| 文字不同步 | 使用 JS 控制精确时机 |
| 背景变化太明显 | 降低颜色差异 |
